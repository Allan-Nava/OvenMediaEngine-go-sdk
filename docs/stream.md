# Stream 


```

Lists all stream names in the Application

Request Example:
GET http://1.2.3.4:8081/v1/vhosts/default/apps/app/streams


```

```
Gets the configuration of the Stream

Request Example:
GET http://1.2.3.4:8081/v1/vhosts/default/apps/app/streams/stream

```
